Woocommerce plugin for EasyCard Israel

== Description ==

EasyCard is one of Israel's leading payment gateway.
Our plugin enables you to accept all credit card types, as well as Bit payments.
Built in support for multiple currencies.

#### Easy Integration

Connecting EasyCard plugin with your WooCommerce store takes just a few simple steps.

#### Key Features

- Credit Card Payment
- Bit Payment
- Integration with Israeli invoicing systems
- Multi currency support
- Webhook Support: Use webhooks to handle real-time payment notifications. Webhook URL: /wc-api/ecng_webhook. Follow the detailed setup guide <a href="https://github.com/EasycardNG/API/blob/main/Webhooks.md" taget="_blank">here</a>.
